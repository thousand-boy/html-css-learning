# 今日学んだこと
### 2026-06-16｜第2章（続き） 表・フォーム・ラベル（2-10〜2-12）

**何をしたか**
- 第2章の続きとして、**表を作る `<table>` 関連タグ**（`tr`・`th`・`td`）の構造と書き方を習得しました。
- `colspan`（横方向）・`rowspan`（縦方向）を使って **複数セルを結合する方法** を学びました。
- HTMLのコード内にメモを残す **コメントアウト**（`<!-- -->`）の書き方と活用方法を学びました。
- `<form>` タグと `<input>` タグを中心に、**フォームの各パーツ**（テキスト入力・ラジオボタン・チェックボックス・送信ボタン・セレクトボックス・複数行テキスト入力）の作り方を学びました。
- `<label>` タグと `for` 属性・`id` 属性を使って **フォームのパーツとラベルを関連付ける方法** を学びました。
- フォームはHTMLで表示部分を作れるが、**実際のデータ送信にはPHPなどプログラムとの連携が必要**であることを理解しました。

**学んだ用語**

| 用語 | やさしい説明 |
|---|---|
| `<table>` | 表全体を囲むタグ |
| `<tr>` | 「Table Row」の略。表の1行を囲むタグ |
| `<th>` | 「Table Header」の略。表の見出しとなるセルを作るタグ |
| `<td>` | 「Table Data」の略。表のデータとなるセルを作るタグ |
| `border="1"` | `<table>` に付ける属性。仮の枠線を表示する（正式な装飾はCSSで行う） |
| `colspan` | 「column span（列をまたぐ）」の略。セルを**横方向**に結合する属性 |
| `rowspan` | 「row span（行をまたぐ）」の略。セルを**縦方向**に結合する属性 |
| コメントアウト | `<!-- -->` で囲んだ部分。ブラウザーに表示されないメモ。コードを一時的に無効化する時にも使える |
| `<form>` | フォーム全体を囲むタグ。テキスト入力欄やボタンなどのパーツをすべてこの中に入れる |
| `action` | `<form>` の属性。入力データを送信する先のページ（ファイル）を指定する |
| `method` | `<form>` の属性。データの転送方法を指定する。主に `get` か `post` を使う |
| `<input>` | フォームのパーツを作るタグ。`type` 属性で表示するパーツの種類を切り替える |
| `type` 属性 | `<input>` タグに付ける属性。入力パーツの種類を指定する（`text`・`radio`・`checkbox`・`submit` など） |
| `placeholder` | `<input>` に付ける属性。入力欄にあらかじめ薄く表示させるヒントテキスト。クリックすると消える |
| `value` | 送信される値や、ボタン上に表示するテキストを指定する属性 |
| `name` | フォームのパーツに付ける名前。同じ `name` 値でラジオボタン・チェックボックスをグループ化できる |
| `checked` | ラジオボタン・チェックボックスを最初から選択された状態にする属性 |
| ラジオボタン | `input type="radio"` で作るパーツ。グループの中から**1つだけ**選択できる |
| チェックボックス | `input type="checkbox"` で作るパーツ。グループの中から**複数**選択できる |
| `<select>` | セレクトボックス（ドロップダウンメニュー）全体を囲むタグ |
| `<option>` | `<select>` の中に入れる各選択肢を囲むタグ |
| `selected` | `<option>` に付ける属性。最初から選択された状態にする |
| `multiple` | `<select>` に付ける属性。Shift または Ctrl キーで複数選択を可能にする |
| `<textarea>` | 複数行のテキストを入力できるパーツを作るタグ。タグで囲んだテキストが初期値として表示される |
| `<label>` | フォームのパーツとラベルテキストを関連付けるタグ。テキスト部分をクリックしてもパーツを選択できるようになる |
| `for` 属性 | `<label>` に付ける属性。関連付けたいパーツの `id` 属性と同じ値を指定する |
| `id` 属性 | フォームのパーツに付ける識別名。同じファイル内で**重複不可**・1か所のみ使用できる |

**学んだ主な構文**

```html
<!-- ===== 2-10 表を作ろう ===== -->

<!-- 基本的な表 -->
<table border="1">          <!-- table：表全体を囲む。border="1" は仮の枠線 -->
  <tr>                      <!-- tr：1行を囲む -->
    <th>プラン名</th>        <!-- th：見出しセル -->
    <th>料金</th>
  </tr>
  <tr>
    <td>わくわくプラン</td>  <!-- td：データセル -->
    <td>月額 2,900円</td>
  </tr>
  <tr>
    <td>にこにこプラン</td>
    <td>月額 1,900円</td>
  </tr>
</table>

<!-- colspan：横方向にセルをつなげる -->
<table border="1">
  <tr>
    <th colspan="2">セル 1+2</th>  <!-- colspan="2" で横2セル分に拡張。thタグを1つ減らす -->
  </tr>
  <tr>
    <td>セル 3</td>
    <td>セル 4</td>
  </tr>
</table>

<!-- rowspan：縦方向にセルをつなげる -->
<table border="1">
  <tr>
    <th>セル 1</th>
    <th>セル 2</th>
  </tr>
  <tr>
    <td rowspan="2">セル 3+5</td>  <!-- rowspan="2" で縦2セル分に拡張。セル5のtdタグを削除 -->
    <td>セル 4</td>
  </tr>
  <tr>
    <td>セル 6</td>
  </tr>
</table>

<!-- コメントアウト -->
<!-- メインコンテンツ部分 ここから -->
<!-- <h2>最新機種の情報</h2> -->  <!-- コードを一時的に非表示にする使い方 -->
```

```html
<!-- ===== 2-11 フォームを作ろう ===== -->

<form action="example.php" method="post" name="contact-form">
  <!-- action=送信先のファイル, method=転送方法, name=フォームの名前 -->

  <!-- 1行テキスト入力欄 -->
  名前：<input type="text">
  <!-- placeholder属性で入力欄にヒントテキストを表示 -->
  名前：<input type="text" placeholder="名字 名前">

  <!-- typeのバリエーション -->
  <input type="search">  <!-- 検索テキスト入力欄 -->
  <input type="email">   <!-- メールアドレス入力欄（ブラウザーが書式チェックしてくれる） -->
  <input type="tel">     <!-- 電話番号入力欄 -->
  <input type="url">     <!-- URL入力欄 -->

  <!-- ラジオボタン：name属性を同じにしてグループ化。1つだけ選択できる -->
  性別：
  <input type="radio" name="gender" value="男"> 男
  <input type="radio" name="gender" value="女" checked> 女   <!-- checked で初期選択 -->
  <input type="radio" name="gender" value="その他"> その他

  <!-- チェックボックス：複数選択可能 -->
  好きな色：
  <input type="checkbox" name="color" value="赤" checked> 赤  <!-- checked で初期選択 -->
  <input type="checkbox" name="color" value="青"> 青
  <input type="checkbox" name="color" value="黄"> 黄
  <input type="checkbox" name="color" value="緑"> 緑

  <!-- 送信ボタン -->
  <input type="submit" value="送信する">   <!-- value属性でボタン上のテキストを指定 -->

  <!-- 画像ボタン -->
  <input type="image" src="images/button.png" alt="送信する">

  <!-- セレクトボックス（ドロップダウン） -->
  血液型：
  <select name="bloodtype">
    <option value="A">A</option>
    <option value="B">B</option>
    <option value="O">O</option>
    <option value="AB">AB</option>
    <option value="不明" selected>不明</option>  <!-- selected で初期選択 -->
  </select>

  <!-- 複数行テキスト入力欄 -->
  <textarea name="message">メッセージを入力</textarea>
  <!-- タグ内テキストが初期値として表示される。消えないのでplaceholderの方が使いやすい -->
  <textarea name="message" placeholder="メッセージを入力"></textarea>

</form>
```

```html
<!-- ===== 2-12 より使いやすいフォームにしよう ===== -->

<!-- label：for属性とid属性の値を同じにして、ラベルとパーツを関連付ける -->
<!-- テキスト部分をクリックしてもチェックボックスを選択できるようになる -->

<input type="checkbox" name="travel" value="日本国内" id="japan">
<label for="japan">日本国内</label>

<input type="checkbox" name="travel" value="ヨーロッパ" id="europe">
<label for="europe">ヨーロッパ</label>

<input type="checkbox" name="travel" value="東南アジア" id="asia">
<label for="asia">東南アジア</label>

<!-- id属性の値はファイル内で一意（重複不可）。1か所にしか使えない -->
```

**できるようになったこと**: `<table>`・`<tr>`・`<th>`・`<td>` を組み合わせて基本的な表を作成できる。`colspan` / `rowspan` でセルを横・縦に結合できる。`<!-- -->` でコードにコメントを残したり一時的に無効化できる。`<form>` タグと `<input>` タグを使ってテキスト入力欄・ラジオボタン・チェックボックス・送信ボタンを作れる。`<select>` と `<option>` でセレクトボックスを作れる。`<textarea>` で複数行テキスト欄を作れる。`<label>` の `for` 属性と `<input>` の `id` 属性で関連付けを行い、クリック範囲を広げた使いやすいフォームを作れる。

**つまずき・気づき**
- `colspan` でセルを横につなげる時は、結合するセル数だけ **`<th>` または `<td>` のタグ数を減らす**（つなげる前の余分なタグを削除しないとレイアウトが崩れる）。
- `rowspan` でセルを縦につなげる時も同様に、縦に並ぶ **不要な `<td>` タグを削除**する必要がある。
- `<tr>` 内のセル数は行ごとに揃えないとレイアウトが崩れる。colspan/rowspan 使用時は特に注意。
- `<textarea>` はタグ内テキストが初期値になるが、入力欄クリックで消えないため使い勝手が悪い。初期表示テキストは **`placeholder` 属性で指定する方が無難**。
- ラジオボタン・チェックボックスは `name` 属性を同じ値にしないとグループにならず、複数選択できてしまう（ラジオボタンの1択制限が効かなくなる）。
- `id` 属性の値はファイル内で **必ず一意**にする。同じ値を複数箇所に使うと `<label>` の関連付けが正しく機能しない。
- `<form>` タグはフォームの**表示部分のみ**を担当する。実際に入力内容を送受信・処理するにはPHPなどサーバーサイドプログラムとの連携が別途必要。
- フォームのデータ送信を簡単に実現したい場合は、メールアドレス宛リンク（`mailto:`）や外部のフォームサービスを使う方法もある。
